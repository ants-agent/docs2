# Roadmap

- Complete a preliminary example demonstrating the use of AI in DeFi.
- Develop an on-chain intelligence system utilizing AI and multiple data sources.
- Simultaneously conduct fundamental research in relevant areas.
- Create an Agent Market for task composition.
- Implement user-oriented agent composition tools.
- Organize an AI Hackathon to encourage developers to design new agents and deploy them within our system, boosting the application of AI agents in various scenarios.
